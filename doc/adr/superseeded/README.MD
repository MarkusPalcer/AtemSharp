This folder contains ADRs which have been superseeded by other ADRs.
The first line of the history should be "- Superseeded {DATE}" and the first line after the heading and before the first paragraph should be 

**Superseeded by [title](link_to_new_adr)**

The superseeding ADR should have the following line at the top

**Superseeds [title](superseeded/this_adr)**
