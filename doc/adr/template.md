# ADR template

Meta:
- The filename is the current date and the title
- The title describes the topic of the ADR

## Context

The first paragraph after the heading (which again is the topic) is a short description of the decision without reasoning.

## Problem

State what problem was to be solved and why it was a problem

## Considered options

Describe some alternatives, including the selected solution

## Decision

Describe why the option was selected over the others

## History

This is a list of state changes of this ADR, newest first.
Each entry is the new state with the date (in the format YYYY-mm-DD),
for example:

- Proposed 2025-10-09
